# 示例文档

这是一个示例markdown文档，用于演示文档系统的功能。

## 功能特点

- 支持markdown语法
- 自动生成sidebar导航
- 文件上传和管理
- 按时间倒序排列

## 使用方法

1. 在首页上传.md文件
2. 文件会自动保存到docs文件夹
3. sidebar会自动更新
4. 可以删除不需要的文件
